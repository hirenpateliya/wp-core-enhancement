# WP Performance Toolkit

WP Performance Toolkit is a lightweight and developer-friendly WordPress plugin designed to optimize performance, clean unnecessary WordPress features, and enhance the admin experience.

## Features

- Disable Gutenberg Editor
- Enable SVG Upload Support
- Duplicate Posts & Pages
- Remove Default WordPress Styles
- Performance Optimization Tweaks
- Custom Admin Settings Panel
- Lightweight & Easy to Use
- Developer Friendly Hooks & Structure

---

## Installation

1. Download the plugin ZIP file
2. Go to WordPress Admin → Plugins → Add New
3. Click Upload Plugin
4. Upload the ZIP file and activate the plugin

OR

Place the plugin folder inside:

wp-content/plugins/
